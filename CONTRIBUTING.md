# Contributing to XELA Themes

Thanks for helping improve XELA Themes. This repository uses ESLint and Prettier. Please follow these steps before opening a PR:

1. Run lint and tests locally:

```bash
npm ci
npm run lint
npm run test
```

2. Format files:

```bash
npm run format
```

3. Keep changes focused and add tests for logic changes.

4. Add a short description and steps to reproduce in PR body.
