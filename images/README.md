# Icon

Place `icon.png` here (128x128 or larger, PNG format) for the VS Code Marketplace listing.

The icon should represent the XELA Themes brand.

For now, you can:
1. Create a simple PNG with the XELA logo
2. Use any of the theme screenshots as a temporary icon
3. Or publish without an icon initially (just comment out the "icon" line in package.json)
