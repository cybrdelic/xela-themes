# XELA Personal Systems Pack

This pack translates the strongest threads visible across the local workspace into a coherent theme family instead of a single all-purpose palette.

## Themes

- `XELA Jarvis Command — Agentic Control`
  Built for AI tooling, agent loops, and command-center work.
- `XELA Runtime Forge — Systems Heat`
  Built for harness work, engine plumbing, build/debug sessions, and operational discipline.
- `XELA Shadernetic Flux — Render Lab`
  Built for WebGPU, shader experiments, simulation, and visual systems.
- `XELA Signal Watch — Reliability Deck`
  Built for watchdogs, perf panels, telemetry, and triage.
- `XELA Portfolio Grid — Editorial Systems`
  Built for portfolio work, product communication, typography, and calm visual hierarchy.
- `XELA Resonance Studio — Creative Nights`
  Built for music, visual experimentation, and expressive late-night making.

## Shared Design Principles

- Clear hierarchy before decoration.
- High contrast with deliberate accent restraint.
- Distinct working modes instead of one compromised default.
- Strong terminal palettes so Windows Terminal and Windows personalization feel first-class, not secondary exports.
- Editorial structure on light surfaces and cinematic depth on dark surfaces.
